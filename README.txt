Stolen Item Notification - Assignment Project

How to run (local):
1. Make sure Python 3 is installed.
2. (Optional) Create and activate a virtual environment.
3. Install Flask: pip install flask
4. From project folder run: python app.py
5. Open http://127.0.0.1:5000/ in your browser.
6. Use the static login -> navigate to User or Admin pages.

Project files are included in this zip.
